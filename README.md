# 專案名稱
世界天氣查詢(串接API作品練習)

# 運行環境
• Node.js
• NPM

# 功能
• 世界天氣查詢(英文)
• 天氣晴、多雲、霧、下雨、暴雨多種天氣狀態
• 查詢不到有警示效果
• keyframe 淡入效果
• Fetch 串接API

• index.js - 串接API、查詢功能
• images - 圖片
